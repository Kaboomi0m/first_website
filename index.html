Jak napisac strone w HTML?

  1. Utworz plik (nazwa strony).html
  2. Otworz notatnik (albo visual studio code)
  3. Zacznij pisac swoja strone uzywajac znacznikow (np. h1,p)
  4. Po skonczonej pracy zapisz plik, ktory nastepnie otworzysz

Jak udoskonalic strone za pomoca css?

  1. Utworz plik (nazwa pliku).css
  2. W pliku html dodaj znacznik link
    ktory bedzie wskazywal na plik css
  3. Zacznij programowac w pliku
